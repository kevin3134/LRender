# LRenderer
This is a software renderer currently based on windows.  
当前为支持window平台的软渲染器。  

![crab example](https://github.com/kevin3134/LRender/blob/main/resource/displayResource/output.gif)


当前功能：  


- [ ] 跨平台  
- [x] 可编程的渲染管线  
- [x] 无外部依赖  
- [x] 窗口和可移动相机  
- [ ] 多线程  
- [x] 可编程的渲染管线  
- [x] 状态机    
- [ ] 齐次裁剪  
- [x] 透视插值  
- [x] 背面剔除   
- [x] 深度测试  
- [ ] 透明混合  
- [ ] 天空盒  
- [ ] 骨骼动画  
- [x] 法线贴图  
- [x] 切线空间法线贴图  
- [ ] 阴影贴图  
- [x] Blinn-Phong  
- [ ] PBR  
- [ ] 环境光照  
- [ ] 材质查看器  
- [ ] MSAA  
- [ ] 多个物体
